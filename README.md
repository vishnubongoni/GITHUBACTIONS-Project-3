# 🛍️ NeoShop E-Commerce (Spring Boot + DevOps Project)

NeoShop is a modern **Spring Boot-based e-commerce UI project** with:
- Pink themed UI 💖
- Product listing with images 🛍️
- Scrolling offers 🎠
- Login/Logout UI 🔐 (frontend simulation)
- Docker containerization 🐳
- CI/CD using GitHub Actions ⚙️

---

# 🚀 Features

- Modern responsive UI
- Product showcase with images
- Auto-scrolling offers section
- Login / Logout UI (frontend demo)
- Spring Boot backend
- Docker support
- GitHub Actions CI/CD pipeline
